5. INTERIOR POINT METHOD
6. QUADRATIC PENALTY METHOD
7. l1-PENALTY METHOD
8. AUGMENTED LAGRANGIAN METHOD (GEOMETRIC FORM)
9. AUGMENTED LAGRANGIAN METHOD
10. LINE-SEARCH
11. PROJECTED GRADIENT METHOD
